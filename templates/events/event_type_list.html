Información sobre Tipos de Eventos
¿Qué es un Tipo de Evento?

Un tipo de evento define una categoría de actividad agrícola (riego, fertilización, cosecha, etc.) que se puede registrar en el sistema de trazabilidad.

Tipos de Eventos Predefinidos

El sistema incluye 10 tipos de eventos predefinidos que cubren las actividades más importantes del ciclo productivo del limón. Estos tipos no se pueden crear, editar o eliminar, pero se pueden activar o desactivar según las necesidades.

¿Cómo activar o desactivar un tipo de evento?

Usa el interruptor (switch) en la columna "Acciones" para activar o desactivar un tipo de evento. Los tipos inactivos no aparecerán disponibles al momento de registrar nuevos eventos.
