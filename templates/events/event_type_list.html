Información sobre Tipos de Eventos
¿Qué es un Tipo de Evento?

Un tipo de evento define una categoría de actividad agrícola (riego, fertilización, cosecha, etc.) y especifica qué datos se deben capturar mediante un esquema JSON.

Esquema JSON

El esquema JSON define los campos personalizados de cada tipo de evento. Por ejemplo, un evento de "Riego" puede tener campos para duración, caudal y presión, mientras que un evento de "Cosecha" tendría campos para cantidad, calidad y destino.

¿Puedo modificar tipos de eventos existentes?

Sí, pero ten cuidado al modificar el esquema de tipos que ya tienen eventos registrados. Los cambios en el esquema no afectan eventos existentes, pero podrían causar inconsistencias.
